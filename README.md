Este repositorio forma parte de mi reto personal **100 días aprendiendo a programar**.

El proyecto sigue creciendo de forma incremental, como ocurre en aplicaciones reales.

---

# Funcionalidades añadidas en este sprint

## ✏️ Editar tareas
Permite modificar el título de una tarea existente.

Esto permite actualizar tareas sin tener que eliminarlas y volver a crearlas.

---

## ✅ Completar todas las tareas
Añade una opción para marcar todas las tareas como completadas en un solo paso.

Útil cuando se quiere cerrar una lista completa de tareas rápidamente.

---

## 🧹 Limpiar tareas completadas
Permite eliminar automáticamente todas las tareas que ya están marcadas como completadas.

Esto ayuda a mantener la lista organizada y limpia.

---

## 📁 Exportar tareas
La aplicación puede exportar todas las tareas a un archivo de texto.

Esto permite guardar o compartir la lista fuera de la aplicación.

---

## 🗑️ Confirmación antes de eliminar
Antes de eliminar una tarea, el programa pide confirmación al usuario.

Esto evita eliminaciones accidentales.
